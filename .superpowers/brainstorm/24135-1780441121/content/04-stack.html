Stack del frontend

¿Con qué tecnología construimos el frontend?

▲

Next.js + shadcn/ui

React · TypeScript · Tailwind

Ventajas

  • Componentes hermosos listos para usar
  • Excelente para dashboards admin
  • TypeScript nativo — mismo lenguaje que el backend
  • Ecosistema enorme, muy documentado
  • Deploy fácil en Vercel

A considerar

  • Requiere saber React básico
La elección más sólida
N

Nuxt 3 + Nuxt UI

Vue 3 · TypeScript · Tailwind

Ventajas

  • Curva de aprendizaje más suave que React
  • Nuxt UI tiene componentes muy pulidos
  • Opinado — menos decisiones que tomar
  • Auto-imports muy convenientes

A considerar

  • Ecosistema más pequeño que React
  • Menos ejemplos de dashboards admin
Buena alternativa Vue
⚛

React + Vite

React · TypeScript · Tailwind

Ventajas

  • Build ultra-rápido con Vite
  • Más simple que Next.js (sin SSR)
  • Total libertad en la arquitectura
  • Ideal si es 100% SPA

A considerar

  • Sin SSR — no ideal para SEO
  • Hay que elegir más librerías manualmente
Más ligero y simple
